#### To run local hardhat node (open another terminal):

``` bash
$ npx hardhat node
$ npx hardhat run --network rinkeby deploy
$ npm run post
$ npm run dInfo
$
$ npm run aInfo

//** look forward for  package.json file **//
```

#### To run bot

``` bash
$ yarn
$ npx hardhat deploy --network xxxx
copy addresses to global constants.ts file
$ yarn post
copy addresses to global constants.ts file
$ yarn bot

```
